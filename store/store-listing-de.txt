== Temperatur und Status von Luxtronik 2.0 Wärmepumpe, Heizung, Warmwasser anzeigen

Zeigt Temperatur und Status von Heizung und Warmwasser einer Wärmepumpe, die mit dem alpha innotec Luxtronik 2.0 Kontrollgerät gesteuert wird.

Sie müssen mit dem gleichen Netzwerk wie das Kontrollgerät verbunden sein (sprich für den Zugriff aus der Ferne ist etwa eine VPN-Verbindung in Ihr Netz zuhause notwendig).

Es werden derzeit nur einige Werte angezeigt, es können keine Einstellungen geändert werden.

Hinweise:
Der Kaufpreis dient als Schutz vor uninformierter Benutzung.
Der Quellcode ist auf GitHub frei verfügbar (siehe Webseite-Schaltfläche).
alpha innotec ist eine Marke der ait–deutschland GmbH.
